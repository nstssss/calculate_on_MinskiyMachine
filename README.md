# Калькулятор на основе машины Минского

Реализация вычислителя арифметических операций с использованием регистровой машины Минского с графическим интерфейсом на PySide6.

## Описание проекта

Машина Минского — это абстрактная вычислительная модель, использующая конечное число регистров и простые операции инкремента и декремента. Данный проект реализует:

- Машину Минского с поддержкой бинарных операций
- Графический интерфейс для взаимодействия
- Операцию сложения целых неотрицательных чисел

## Основные классы

### MinskyMachine
Центральный класс, реализующий машину Минского:
- Управление регистрами
- Загрузка и выполнение программ
- Методы для арифметических операций

### Register
Регистр машины, содержащий:
- Ленту с бинарным представлением числа
- Операции инкремента и декремента

### Tape
Бесконечная лента для хранения бинарных чисел:
- Поддержка операций в бинарном представлении
- Автоматическое расширение при необходимости

### Transition
Инструкция программы машины Минского:
- Определяет операцию и переходы между инструкциями

## Установка и запуск

1. Клонируйте репозиторий:
```bash
git clone https://github.com/username/minsky-calculator.git
cd minsky-calculator
```

2. Установка зависимостей:
```bash
pip install -r requirements.txt
```

3. Инструкция по запуску:
```bash
python minsky_app.py
```


Запуск тестов:
```bash
python test_minsky.py
```

## Предварительные требования

- Python 3.8+
- PySide6
- pytest (для тестирования)
